# 🐾 PetMatch — Tinder для животных

PetMatch — это платформа, которая помогает животным найти любящих хозяев, а людям — верного друга.  
Приюты и частные лица создают анкеты животных, а пользователи выбирают их свайпами вправо/влево. При взаимной симпатии открывается чат.

---

## 🚀 Функционал

- ✅ Лента с карточками животных (свайп вправо — лайк, влево — пропуск)
- ✅ Страница животного с фото, описанием, контактами
- ✅ Создание и редактирование анкеты животного
- ✅ Взаимные лайки → автоматическое открытие чата
- ✅ Чат 
- ✅ iOS-приложение + веб-сайт + Telegram-бот (анонимное общение)

---
